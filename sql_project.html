AtliQ Mart Data Analyst Using SQL

This project aims to analyze the performance of promotions conducted by Atlīq Mart during Diwali 2023 and Sakranti 2024 on their Atlīq branded products across their 50 supermarkets in the southern region of India. The goal is to identify which promotions were successful and which ones were not, providing insights for informed decision-making in future promotional activities.

Telangana Tourism Data Analysis using SQL

Telangana's rich cultural heritage, history, and topography have endowed the state with a variety of tourist destinations, ranging from waterfalls and hills to temples and forts. This project aims to provide insights into which city has the highest tourism capability and which one has the least, as well as strategies for increasing tourism in cities with potential for growth.
